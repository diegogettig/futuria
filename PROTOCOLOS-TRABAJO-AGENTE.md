# FUTURIA Institute — Protocolos de Trabajo del Agente

## 1. Uso exclusivo de este chat (chat operativo)

Este chat está reservado exclusivamente para:
- Puesta en funcionamiento operativo de FUTURIA Institute y todas sus verticales
- Ejecución de tareas concretas (creación de archivos, configuración de dominios, despliegues, outreach)
- Coordinación de entregables tangibles
- Seguimiento de avances y bloqueos operativos

**Regla:**
- Cualquier discusión estratégica de alto nivel, definición de líneas de pensamiento, o debate sobre escenarios e ideas debe derivarse a los chats especializados creados con los prompts 1 y 2.
- Este chat NO es para debates filosóficos abstractos ni para discusiones sobre metodología PRISM en profundidad (eso va al chat de escenarios).

---

## 2. Jerarquía de espacios de trabajo

```
Chat Operativo (este)
    ├── Ejecución
    ├── Configuración
    ├── Despliegue
    ├── Outreach
    └── Logística

Chat Gerencia Estratégica (Prompt 1)
    ├── Alineamiento visión/misión/valores
    ├── Revisión de avances y objetivos
    ├── Definición de escenarios de trabajo
    ├── Estrategia general
    ├── Estructura organizacional
    ├── Cashflow y modelos de negocio
    └── Monetización

Chat Laboratorio de Escenarios (Prompt 2)
    ├── Discusión de escenarios
    ├── Ideas sueltas
    ├── Líneas de pensamiento
    ├── Input para investigaciones
    ├── Formas de mostrar ideas
    └── Base para investigaciones detalladas del FI
```

---

## 3. Formato de entregables en este chat

Todo archivo generado debe:
- Guardarse en la ruta canónica del proyecto (`C:\Users\Diego\Projects\FUTURIA\...`)
- Incluir encabezado con fecha, autor y estado
- Ser ejecutable (no solo teórico)

Todo enlace o URL generada debe:
- Ser verificada inmediatamente (web_extract o browser_vision)
- Ser reportada al usuario con estado funcional/roto

---

## 4. Regla de confirmación de avances

Después de cada bloque de ejecución (creación de archivos, configuración, outreach), el agente debe:
1. Listar lo ejecutado
2. Indicar el próximo paso operativo
3. Preguntar al usuario por dónde continuar

---

## 5. Tratamiento de información sensible

- Todo lo compartido en este chat se considera CONFIDENCIAL
- No se comparten credenciales, claves ni datos sensibles en archivos
- Los archivos generados no deben contener información personal sensible del usuario
- Cualquier captura de pantalla compartida por el usuario se trata como información confidencial

---

## 6. Cadencia de trabajo

- Este chat: puede ser continuo, respuesta por respuesta
- Chat Gerencia Estratégica: se espera interacción estructurada, con reportes periódicos
- Chat Laboratorio de Escenarios: puede ser más libre, exploratorio
