## AGENTES IA: ORDEN DE IMPLEMENTACIÓN

### Fase inmediata (usar herramientas existentes, no desarrollar)
1. **VIGÍA (MVP):** RSS feeds + web scraping con Python/Feedparser. No necesita IA en v1.
2. **ANALISTA (MVP):** LLM (Yo mismo/Hermes) haciendo síntesis de inputs. Automatizable via cron jobs.
3. **AUDITOR (MVP):** Prompt adversarial aplicado a outputs del ANALISTA.

### Fase posterior (inversión de desarrollo)
4. **NARRADOR** y **ARQUITECTO** cuando haya masa crítica de investigación
5. **CURADOR** cuando el graph de conocimiento justifique la inversión
